XML PHP

XML Parser

XML = language to structure data to share across website
XML is like HTML but tags are defined by the user

There are two XML Parser in PHP

Tree Based Parser

Hold docs in memory and transfrom xml into tree structure
better for small XML document

Example of Tree Based Parser
  • SimpleXML
  • DOM

Event Based Parser

Read one node at a time allowing user to interact in real-time
better for larger XML and a lot faster with lesser memoery consuming

Example of Event Based Parser
  • XMLReader
  • XML Expat Parser
